# 新增 INSERT INTO 语句

MySQL 表中使用 INSERT INTO SQL语句来插入数据。

# 语法

INSERT INTO table_name ( field1, field2,...fieldN )
VALUES( value1, value2,...valueN );
1
2

TIP

如果数据是字符型，必须使用单引号或者双引号，如："value"。

# 实例

以下我们将使用 SQL INSERT INTO 语句向 MySQL 数据表 Websites 插入数据

INSERT INTO Websites
  (name, url, alexa, country)
  VALUES
  ("京东", "https://www.jd.com/", 200, "CN");

# 输出
Query OK, 1 row affected (0.03 sec)

# 查询结果
mysql> SELECT * FROM Websites;

# 查询 SELECT 语句

MySQL 表中使用 SELECT 语句用于从数据库中选取数据，结果被存储在一个结果表中，称为结果集。

# 语法

SELECT column1, column2, ...
FROM table_name;

# or

SELECT * FROM table_name;
1
2
3
4
5
6

TIP

  • column1, column2, ...：要选择的字段名称，可以为多个字段。如果不指定字段名称，则会选择所有字段。
  • table_name：要查询的表名称。

# 实例

下面的 SQL 语句从 "Websites" 表中选取 "name" 和 "country" 列：

# 查询语句
SELECT name,country FROM Websites;

# 输出

| name         | country |
+--------------+---------+
| Google       | USA     |
| 淘宝         | CN      |
| 菜鸟教程     | CN      |
| 微博         | CN      |
| Facebook     | USA     |
+--------------+---------+
5 rows in set (0.01 sec)

# 更新 UPDATE 语句

如果我们需要修改或更新 MySQL 中的数据，我们可以使用 SQL UPDATE 命令来操作。

# 语法

UPDATE table_name SET field1=new-value1, field2=new-value2
[WHERE Clause]
1
2

TIP

  • 你可以同时更新一个或多个字段
  • 你可以在 WHERE 子句中指定任何条件
  • 你可以在一个单独表中同时更新数据

# 实例

以下实例将更新数据表中 id 为 5 的 name 字段值：


UPDATE Websites SET name="脸书" where id=5;

mysql> UPDATE Websites SET name="脸书" where id=5;

# 输出
Query OK, 1 row affected (0.01 sec)
Rows matched: 1  Changed: 1  Warnings: 0

# 查询结果
mysql> SELECT * FROM Websites;

# 删除 DELETE 语句

你可以使用 SQL 的 DELETE FROM 命令来删除 MySQL 数据表中的记录。

# 语法

DELETE FROM table_name [WHERE Clause]
1

TIP

  • 如果没有指定 WHERE 子句，MySQL 表中的所有记录将被删除。
  • 你可以在 WHERE 子句中指定任何条件
  • 您可以在单个表中一次性删除记录。

# 实例

以下实例将删除 Websites 表中 id 为 6 的记录：

DELETE FROM Websites where id = 6;

# 输出
mysql> DELETE FROM Websites where id = 6;
Query OK, 1 row affected (0.01 sec)

mysql> select * from Websites;
+----+--------------+---------------------------+-------+---------+
