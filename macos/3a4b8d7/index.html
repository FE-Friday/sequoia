NVM（Node Version Manager）是一个用于在基于 Linux 系统上安装和管理 Node.js 的 shell 脚本。

MacOS 用户可以使用 homebrew 来安装 NVM。

本教程帮助你在macOS系统上安装NVM并管理 Nodej.is 版本。

WARNING

注意：本教程需要安装好 Homebrew

# 安装

brew install nvm
1

执行结果如下图，根据提示配置环境变量。

# 配置环境变量

# 创建文件
vi .zshrc

# 写入下面的内容
export NVM_DIR="$HOME/.nvm"
  [ -s "/opt/homebrew/opt/nvm/nvm.sh" ] && \. "/opt/homebrew/opt/nvm/nvm.sh"  # This loads nvm
  [ -s "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm" ] && \. "/opt/homebrew/opt/nvm/etc/bash_completion.d/nvm"

# 更新配置
source ~/.zshrc

# NVM 常用命令


nvm --help # 查看帮助信息

nvm ls-remote # 查看所有可用版本

nvm install <version> # 安装指定版本的 Node

nvm install --lts # 安装最新的长期支持版本

nvm install stable  # 安装最新稳定版 node

nvm uninstall <version>  # 删除已安装的指定版本，语法与install类似

nvm use <version> # 切换使用指定的版本node

nvm alias default <version> # 设置默认版本

nvm run <version> --version # 也可以直接运行指定版本

nvm which <version> # 查看安装路径
